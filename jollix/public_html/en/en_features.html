Deutsche Version

Jollix 0.3 Features:

  • based on Gentoo Linux
  • LiveCD - No hard disk installation necessary
  • optimized for i686 architecture
  • Kudzu hardware detection
  • Linux Kernel: gentoo-dev-sources 2.6.5-r1
  • XOrg-X11 6.7.0
  • KDE 3.2.2
  • Many multimedia applications (audacity, xmms, xine, kino, kdenlive, K3B)
  • WLAN and bluetooth support
  • list of installed applications (in a new window)
