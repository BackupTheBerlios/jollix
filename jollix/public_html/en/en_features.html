Deutsche Version

Jollix 0.2 Features:

  • based on Gentoo Linux
  • no hard disk installation necessary (LiveCD)
  • optimized for i686 architecture
  • supports nVidia und ATI-based graphics cards with full 3D acceleration, other graphics cards use Vesa mode
  • Kernel: Gentoo 2.4.20-gaming-sources
  • KDE 3.1.4
  • Many multimedia applications (xmms, mplayer, xine u.a.)
  • ISDN-support
  • Fritz!Card DSL
  • Wine Build 20030911
  • XQF, game-server browser
  • Cheathing Death
  • Half-Life-, Public-Enemy- or Counter-Strike hard disk installations can be used directly from the LiveCD!
  • Many bugfixes + current versions of installed software
  • Installation instructions for experienced linux users

  • list of packets (in new window)
