0. Was ist jollix 0.1?
Jollix ist eine einfach zu installierende und komplett freie Linux Distribution
für die x86 Architektur, die auf Gentoo Linux basiert. Mit jollix soll
haupstächlich dem Gamer und Multimedia-Enthusiast der vollständige Umstieg
von MS-Windows auf Linux erleichtert werden. Die Version 0.1 ist die erste
Beta-Release zum Testen von Counterstrike unter Linux. Weitere Spiele und
die Möglichkeit zur Installation auf Deiner Festplatte folgen.


	
1. Netzwerk einrichten
Über Desktop-Button "Netzwerk einrichten" das lokale
LAN-Netzwerk einrichten. "eth0" bezeichnet die erste Netzwerkkarte im Rechner,
eth1 die zweite usw. Befindet sich nur eine Netzwerkkarte im Rechner, kann
die Vorgabe "eth0" übernommen werden.
		
2. ADSL einrichten
Die ADSL Verbindung wird eingerichtet über "ADSL einrichten".
Es öffnet sich ein Fenster, in dem das root-Passwort abgefragt wird.
Danach wirst Du durch die ADSL Einrichtung geführt.
	
Einige Anmerkungen:
a) Solltest Du T-Online DSL Kunde sein, dann setzt sich Dein Benutzername 
   zusammen aus:
1. Anschlusskennung
2. Zugehöriger T-Online Nummer
3. Suffix #0001
. @t-online.de
z.B. 111111111111222222222222#0001@t-online.de
b) Solltest Du mehr als eine Netzwerkkarte haben, gilt das gleiche wie unter 
   2. "Netzwerk einrichten".
   Die erste Karte in Deinem Rechner ist "eth0", die zweite "eth1" usw.

Sobald die Einrichtung abgeschlossen ist, kannst Du mit
der Desktop-Verknüpfung "ADSL starten" die Verbindung aufrufen.

3. Counterstrike einrichten
Über den Desktop-Button "Counterstrike einrichten" eine Desktop-Verknüpfung
mit der Counterstrike Installation auf Deiner Festplatte einrichten.
Im erscheinenden Dialog den Installationsordner von Counterstrike angeben,
in dem sich hl.exe bzw. cstrike.exe befindet. Deine Festplatten werden in
dem Dialog so oder ähnlich erscheinen: ide/host0/bus0/target0/lun0/part1
(Alle Partitionen sind schreibgeschützt und können von jollix nicht verändert
werden!)

4. Passwörter
Damit es fürs erste einfacher ist, haben wir Passwörter für die beiden
eingerichteten Benutzer vorgegeben. Der Benutzer "root" ist vergleichbar mit dem
"Administrator" unter Windows, daher sollte man nur dann "root" benutzen, wenn es
von jollix verlangt wird. Der Benutzer "user" ist der eingerichtete
Standardbenutzer, mit dem man spielen und surfen sollte. Wenn Du einen 
Internetzugang in jollix einrichten willst (Netzwerk oder ADSL), dann solltest Du
die beiden Passwörter vorher ändern. Nach einem Neustart der jollix LiveCD sind 
die beiden Passwörter allerdings wieder zurückgesetzt, da sie nicht dauerhaft 
auf der CD gespeichert werden können.
Das root-Passwort und das user-Passwort ändert man am einfachsten, indem man im
K-Menu unter Programme -> System -> "KUser" aufruft und dort in der Liste den 
Benutzer "user" bzw. "root" per Doppelklick auswählt.

Das root-Passwort lautet: jollix

Das user-Passwort lautet: user

What is Gentoo?

From www.gentoo.org:
"We produce Gentoo Linux, a special flavor of Linux that can be automatically optimized and customized for just about any application or need. Extreme performance, configurability and a top-notch user and developer community are all hallmarks of the Gentoo experience. Thanks to a technology called Portage, Gentoo Linux can become an ideal secure server, development workstation, professional desktop, gaming system, embedded solution or something else -- whatever you need it to be. Because of its near-unlimited adaptability, we call Gentoo Linux a metadistribution."

Can I check out jollix without repartition or formatting my harddisk?

YES, you can. Jollix can be started "read-only" from a CD (as a LiveCD). The installed system is not touched. Programs installed under windows can be started (e.g. Counterstrike)

What is wine?

From Wine: "Wine is an Open Source implementation of the Windows API on top of X and Unix. Think of Wine as a Windows compatibility layer. Wine does not require Microsoft Windows, as it is a completely alternative implementation consisting of 100% Microsoft-free code, but it can optionally use native system DLLs if they are available. Wine provides both a development toolkit (Winelib) for porting Windows sources to Unix and a program loader, allowing many unmodified Windows binaries to run on x86-based Unixes, including Linux, FreeBSD, and Solaris."
but not all windows programs and games ar guaranteed to run with wine.

Was ist KDE?

From www.kde.org "KDE is a network transparent contemporary desktop environment for UNIX workstations. KDE seeks to fill the need for an easy to use desktop for Unix workstations, similar to the desktop environments found under the MacOS or Microsoft Windows. We believe that the UNIX operating system is the best operating system available today. In fact UNIX has been the undisputed choice of the information technology professional for many years. When it comes to stability, scalability and openness there is no competition to UNIX. However, the lack of an easy to use contemporary desktop environment for UNIX has prevented UNIX from finding its way onto the desktops of the typical computer user in offices and homes. UNIX dominates the server market and is the preferred computing platform for computing professional and scientists. Without UNIX the internet would not be. But UNIX did not address the needs of the average computer user. This fact is particularly unfortunate since a number of implementations of UNIX (Debian GNU/Linux, FreeBSD, NetBSD etc.) are freely available on the internet. All of which are of exceptional quality and stability."
