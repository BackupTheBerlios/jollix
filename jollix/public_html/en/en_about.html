What is jollix?

Jollix ia a LiveCD based on Gentoo Linux Jollix boots directly from CD without touching the hard disk. A fast and easy-to-install version of jollix will be available in a later release. In the current version there are installation instructions for more experienced linux users. Once jollix is installed on a hard disk, it can be updated via portage, the packet manager of Gentoo linux. In fact, jollix IS gentoo at this point (somehow preconfigured).
Jollix aimes at the gamer and multimedia-enthusiast, who prefers Linux as a stable and safe platform.


Included in the distribution is the desktop environment KDE with many useful applications (mostly internet and multimedia).
Also part of jollix is wine, a program which allows the operation of DOS and MS Windows programs (Windows 3.x and Win32 executables) on Linux. Especially games like Counter-Strike (in general: "Half-Life-based") are playable under Linux with wine.
If you successfully installed jollix and want more features/applications, you can do this by simply using the gentoo packet managing tool portage to install new/upgrade applications. Deinstallation packets is - of course - possible too. A simple graphical installation is currently not possible (but planned). Experienced Users can try to install jollix on their hard disk following this instructions (german only so far! we will translate to english ASAP).

Was is jollix NOT?

Jollix has nothing to do with jolix, the freeware Port of the BSD Net/2 system to the Intel i386 architecture by Bill and Lynne Greer Jolitz. ;-)
