22.10.03 - Jollix 0.2 available.
Download here.
Version 0.2 is the second beta-release of jollix. The most important enhancements are: supports ATI Radeon graphics cards, ISDN-cards, Fritz!Card DSL by AVM. Many internet and multimedia applications are part of the software collection, e.g. kopete (ICQ-client), KVIRC, XQF (gameserver browser), Kaffeine (mediaplayer) and TVTime (TV-application). A detailed overview is available here. Experienced GNU/linux users might try to install jollix on their hard disk. But the installation instructions are in german only so far. This will change soon. We plan to develop a graphical installation program for a later version. Jollix is optimized for the i686 architecture; its core is the gaming-sources-2.4.20 kernel from gentoo linux. NVIDIA (TNT- Geforce 5900 FX) and ATI Radeon graphics cards (8500-9800) are supported with full OpenGL 3D acceleration. the size of the ISO-image of the jollix LiveCD is 506MB.

06.08.03 - Gentoo 1.4 released.
Today Gentoo Linux 1.4 was released as a LiveCD. Work on jollix is going to be finished soon.
"The Network-Integrated Multimedia Middleware (NMM) provides an architecture that allows to create distributed multimedia application easily: local and remote multimedia devices or software components can be controlled transparently and integrated into a common multimedia processing flow graph. NMM runs under Linux, is implemented in C++, and distributed under the LGPL." (cited from http://graphics.cs.uni-sb.de/NMM/index.html. Go here for details)
The CVS archive of the NMM project contains the Multimedia-Box (MMBox), an NMM application that can be described as an extensible Home Entertainement System (like freevo for example). We are working on a MMBox-version of jollix. This LiveCD will boot directly into MMBox. Furthermore this LiveCD might be used as a "serverregistry-enabling" CD, i.e. via boot parameter "jollix doserverregistry" the PC acts as the central registry for the middleware instead starting up the MMBox. See documentaion of MMBox for details about this application and the serverregistry.
