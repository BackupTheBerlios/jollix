Was ist jollix?

Jollix ist derzeit eine LiveCD die auf Gentoo Linux basiert. D.h. man bootet von CD und benutzt das System wie Knoppix nur von CD aus ohne die Festplatte beschreiben zu könnnen. Als einfach zu installierende und komplett freie Linux Distribution für die x86 Architektur, wird es jollix in einer späteren Version einmal geben. Danach hat man de facto Gentoo Linux auf der Platte. und kann dieses System auch beliebig erweitern, updaten, usw.
Mit jollix soll hauptsächlich dem Gamer und Multimedia-Enthusiast der vollständige Umstieg von MS-Windows auf Linux erleichtert werden. Zum Ausprobieren kann man jollix - ganz wie Knoppix - direkt von einer CD booten, ohne die Festplatte mit bereits bestehenden Partitionen anzurühren. Dazu lädt man sich das ISO Image einfach herunter, brennt es auf eine CD und bootet damit den PC neu. Soll das bestehende System ersetzt werden, kann dieselbe CD (irgendwann einmal ;-) zum Installieren eines vollwertigen und hochoptimierten Linux Systems verwendet werden.

Die Installation geschieht fast vollautomatisch, da sie bestimmte Hardwarekomponenten (Athlon CPU, nVidia oder ATi Grafiksystem, die gängigsten Soundkarten usw.) vorraussetzt. Für diese Umgebung ist jollix dann aber optimal konfiguriert und quasi "out-of-the-box" benutzbar!

Neben der Desktop-Umgebung KDE samt einiger nützlicher Programme (Multimedia-Anwendungen, CD-Brennprogramm usw.) ist auch wine enthalten. Wine ist ein Programmpaket, mit dem auch MS-Windows Programme unter Linux laufen. Dadurch sind insbesondere auch Spiele wie Counter-Strike unter Linux benutzbar, für die es (noch) keine Linux-Version gibt.

Sollte der Funktionsumfang irdendwann nicht mehr ausreichend sein, kann das System einfach erweitert werden. D.h. zuätzliche Programmpakete können - ganz wie bei Gentoo Linux - installiert und auch wieder deinstalliert werden. Dafür wird das von den gentoo-Machern entwickelte Portage System gleich mit installiert. Jollix vereint so alle Vorteile der High-End-Geek-Linux Distribution Gentoo mit den Bedürfnissen und Ansprüchen eines MS-Windows Nutzers, den bisher nur das Zocken seines Lieblings-Shooters vom Umstieg auf Linux abgehalten hat.

Was ist jollix nicht?

Jollix hat nichts mit jolix, dem Freeware Port des BSD Net/2 Systems auf die Intel i386 Architektur von Bill und Lynne Greer Jolitz zu tun. ;-)
