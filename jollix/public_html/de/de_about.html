Was ist jollix?

Jollix ist derzeit eine LiveCD die auf Gentoo Linux basiert. D.h. man bootet von CD und benutzt das System wie Knoppix nur von CD aus ohne die Festplatte beschreiben zu könnnen. Eine Version zur dauerhaften Installation auf Festplatte wird es irgendwann später auch einmal geben. Danach hat man de facto Gentoo Linux auf der Platte, das man über das Ports-System von Gentoo dann beliebig erweitern und updaten kann.
Mit Jollix soll hauptsächlich der Gamer und Multimedia-Enthusiast angesprochen werden, der die Linux Plattform bevorzugen will. Zum Ausprobieren kann man jollix mit einer automatischen Hardware-Erkennung direkt von einer CD booten, ohne die Festplatte mit bereits bestehenden Partitionen zu verändern. Dazu lädt man sich einfach das ISO Image herunter, brennt es auf eine CD und bootet damit den PC neu.

Neben der Desktop-Umgebung KDE samt einiger nützlicher Programme (v.a. Internet- und Multimedia-Anwendungen) ist auch Wine enthalten. Wine ist ein Programmpaket, mit dem auch einige MS-Windows Programme unter Linux laufen. Dadurch sind insbesondere Spiele wie Counter-Strike unter Linux lauffähig, für die es (noch) keine eigene Linux-Version gibt (und vielleicht auch nie geben wird).

Sollte nach einer Festplatteninstallation der Funktionsumfang irgendwann nicht mehr ausreichen, kann das System einfach erweitert werden. D.h. zuätzliche Programmpakete können - ganz wie bei Gentoo Linux - installiert und auch wieder deinstalliert werden. Dafür wird das von den Gentoo-Machern entwickelte System namens Portage gleich mit installiert. Eine richtige Installtion ist derzeit noch nicht möglich, jedoch ab einer der kommenden Versionen (spätestens 1.0 ;-)

Was ist jollix nicht?

Jollix hat nichts mit jolix, dem Freeware Port des BSD Net/2 Systems auf die Intel i386 Architektur von Bill und Lynne Greer Jolitz zu tun. ;-)
