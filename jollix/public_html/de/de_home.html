English Version

Jollix ist eine freie GNU/Linux Distribution, die direkt von CD läuft (LiveCD). Jollix ist ein speziell für Spiele und Multimedia vorkonfiguriertes Gentoo Linux, das automatisch gängige PC-Hardware erkennen und konfigurieren kann. Die LiveCD startet automatisch eine grafische Oberfläche (XFree und KDE), so dass direkt alle wichtigen Anwendungen benutzt werden können. Mehr Informationen gibt es hier.

News & Ankündigungen:

22.10.03 - Jollix 0.2 ist fertig und kann hier heruntergeladen werden.
Die Version 0.2 ist die zweite Beta-Release von jollix. Die wichtigsten Erweiterungen gegenüber der Version 0.1.1 sind die Unterstützung von ATI Radeon Grafikkarten, ISDN-Karten und der FritzCard DSL von AVM. Jollix bringt viele Internet- und Multimedia Anwendungen gleich mit, u.a. Kopete (ICQ-Klient), KVIRC, XQF (Gameserver Browser), Kaffeine (Mediaplayer) und TVTime (Fernsehapplikation). Eine ausführliche Liste der wichtigsten Programme ist hier zu finden. Eine Festplatteninstallation von jollix ist für erfahrene Benutzer mit Hilfe einer Anleitung nun ebenfalls möglich. Für eine spätere Version von jollix ist ein einfaches grafisches Installationsprogramm geplant. Jollix ist optimiert für die i686 Architektur und bringt einen auf Spiele optimierten Linux Kernel mit (gaming-sources). NVIDIA (TNT- Geforce 5900 FX) und ATI Radeon Karten (8500-9800) werden mit voller OpenGL 3D Beschleunigung unterstützt. Das ISO-Image zum Brennen der jollix CD ist 506 MB groß.

05.09.03 - Jollix 0.2-beta steht in Kürze zum Download bereit.
Die Version 0.2-beta von jollix ist fertig und wird - wenn alles klappt - im Laufe des Tages für die registrierten Beta-Tester bei Berlios zum Download bereit stehen. Nach einer anvisierten 2-wöchigen Testphase soll dann die allgemeine Veröffentlichung von jollix 0.2 folgen.
Die wichtigsten Neuerungen sind:
  • ATI Grafikkarten 3D-Support
  • AVM FritzCardDSL
  • ISDN (capi4linux)
  • Digital Kamera Hotplug
  • XQF Gamebrowser
  • TV Karten Unterstützung
Weitere Einzelheiten sowie bekannte Fehler, workarounds + hacks werden im Forum und hier in den nächsten Tagen bekannt gegeben. Happy Testing ;-)

06.08.03 - Gentoo 1.4 erschienen. Seit heute ist Gentoo Linux 1.4 als LiveCD zum Download freigegeben. Die Arbeit an Jollix 0.2 geht deshalb langsam in die Betatest-Phase über.
Die NETWORK-INTEGRATED MULTIMEDIA MIDDLEWARE für Linux (NMM) ist ein Projekt an der Universität des Saarlandes, das eine Infrastruktur zwischen vernetzten Multimedia-Geräten (PCs, Kameras, Handhelds usw.) etabliert. Enthalten im CVS Archiv ist u.a. die Multimedia-Box, eine NMM-Anwendung, die ein erweiterbares Home Entertainment System bereit stellt (ähnlich wie freevo z.B.). In der Zusammenarbeit mit dem NMM-Team ist geplant, für die Multimedia-Box eine eigene Jollix Version zu schaffen, so dass dann die Mutimedia-Box direkt von der bootfähigen CD-ROM gestartet werden kann.
