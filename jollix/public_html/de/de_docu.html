0. Was ist jollix 0.1?
Jollix ist derzeit eine LiveCD die auf Gentoo Linux basiert. D.h. man bootet	
von CD und benutzt das System wie knoppix nur von CD aus ohne die Festplatte 
beschreiben zu könnnen
Als einfach zu installierende und komplett freie Linux Distribution
für die x86 Architektur, wird es jollix in einer späteren Version einmal geben.
Danach hat man de facto Gentoo Linux auf der Platte. und kann dieses System
auch beliebig erweitern, updaten, usw.
Mit jollix soll haupstächlich dem Gamer und Multimedia-Enthusiast der 
vollständige Umstieg von MS-Windows auf Linux erleichtert werden. 
Die Version 0.1 ist die erste Beta-Release zum Testen von Counterstrike 
unter Linux. Weitere Spiele und die Möglichkeit zur Installation auf 
Deiner Festplatte folgen.


	
1. Netzwerk einrichten
Über Desktop-Button "Netzwerk einrichten" das lokale
LAN-Netzwerk einrichten. "eth0" bezeichnet die erste Netzwerkkarte im Rechner,
eth1 die zweite usw. Befindet sich nur eine Netzwerkkarte im Rechner, kann
die Vorgabe "eth0" übernommen werden.
		
2. ADSL einrichten
Die ADSL Verbindung wird eingerichtet über "ADSL einrichten".
Es öffnet sich ein Fenster, in dem das root-Passwort abgefragt wird.
Danach wirst Du durch die ADSL Einrichtung geführt.
	
Einige Anmerkungen:
a) Solltest Du T-Online DSL Kunde sein, dann setzt sich Dein Benutzername 
   zusammen aus:
1. Anschlusskennung
2. Zugehöriger T-Online Nummer
3. Suffix #0001
. @t-online.de
z.B. 111111111111222222222222#0001@t-online.de
b) Solltest Du mehr als eine Netzwerkkarte haben, gilt das gleiche wie unter 
   2. "Netzwerk einrichten".
   Die erste Karte in Deinem Rechner ist "eth0", die zweite "eth1" usw.

Sobald die Einrichtung abgeschlossen ist, kannst Du mit
der Desktop-Verknüpfung "ADSL starten" die Verbindung aufrufen.

3. Counterstrike einrichten
Über den Desktop-Button "Counterstrike einrichten" eine Desktop-Verknüpfung
mit der Counterstrike Installation auf Deiner Festplatte einrichten.
Im erscheinenden Dialog den Installationsordner von Counterstrike angeben,
in dem sich hl.exe bzw. cstrike.exe befindet. Deine Festplatten werden in
dem Dialog so oder ähnlich erscheinen: ide/host0/bus0/target0/lun0/part1
(Alle Partitionen sind schreibgeschützt und können von jollix nicht verändert
werden!)

4. Passwörter
Damit es fürs erste einfacher ist, haben wir Passwörter für die beiden
eingerichteten Benutzer vorgegeben. Der Benutzer "root" ist vergleichbar mit dem
"Administrator" unter Windows, daher sollte man nur dann "root" benutzen, wenn es
von jollix verlangt wird. Der Benutzer "user" ist der eingerichtete
Standardbenutzer, mit dem man spielen und surfen sollte. Wenn Du einen 
Internetzugang in jollix einrichten willst (Netzwerk oder ADSL), dann solltest Du
die beiden Passwörter vorher ändern. Nach einem Neustart der jollix LiveCD sind 
die beiden Passwörter allerdings wieder zurückgesetzt, da sie nicht dauerhaft 
auf der CD gespeichert werden können.
Das root-Passwort und das user-Passwort ändert man am einfachsten, indem man im
K-Menu unter Programme -> System -> "KUser" aufruft und dort in der Liste den 
Benutzer "user" bzw. "root" per Doppelklick auswählt.

Das root-Passwort lautet: jollix

Das user-Passwort lautet: user

FAQ

Was ist jollix?

Hier klicken.

Was ist Gentoo?

Gentoo Linux ist eine vielseitige und schnelle, komplett freie Linux Distribution, die für Linux Power-User gedacht ist. Im Gegensatz zu anderen Distributionen hat Gentoo Linux ein fortschrittliches Paketverwaltungs-System namens Portage. Portage ist ein Ports-System in der Tradition von BSD-Ports. Es ist in Python realisiert und bringt einige zusätzliche Features mit, darunter auch ein fein granuliertes Management von Abhängigkeiten, die Möglichkeit, "fake"-Installationen als OpenBSD durchzuführen, ein Pfad-Sandboxing, sicheres "Unmerging", System-Profile, virtuelle Pakete und ein Konfigurationsdatei-Management.
Gentoo Linux bildet die Basis von jollix und ist auf www.gentoo.org sehr gut dokumentiert.

Kann ich jollix gefahrlos nur mal eben ausprobieren?

Ja. Jollix kann "nur-lesend" direkt von CD-ROM gestartet werden. Dabei wird das bestehende System nicht verändert. Unter Windows installierte Programme können teilweise sogar gestartet werden, z.B. Counter-Strike.

Was passiert, wenn ich jollix installiere?

Die Partitionstabelle der ersten Festplatte wird neu geschrieben, um nach einer Formatierung der gesamten Platte, jollix dort zu installieren D.h. alles was sich auf der Festplatte befindet, Betriebsystem, Anwwendungsprogramme, persönliche Daten usw, wird unwiederbringlich gelöscht. Jollix ist in dieser Hinsicht kompromisslos. Es werden keine Partitionen vergrössert oder verkleinert wie bei anderen Distributionen. Eine eventuell vorhandene zweite Festplatte wird nicht angetastet.

Was ist wine?

Wine ist eine Implementierung der Windows Win32 und Win16 APIs auf Basis des Fenstermanagers X und Unix. Wine stellt unter Linux eine Umgebung zur Verfügung, in der Windows-Programme (.exe Dateien) auch unter Linux laufen. Wine benötigt keine Installation von Windows, da es eine hundertprozentige Unabhängigkeit vom Microsoft Code hat. Optional kann es die System DLL`s zur besseren Unterstützung der Windowsprogramme nutzen. Wine kommt mit dem Quellcode, mit Dokumentationen, Beispielen und es ist frei einsetzbar.
Nicht alle Windows-Programme, insbesondere neuere, sind jedoch auch garantiert lauffähig. So sind nicht alle Funktionen der neuesten Windows Versionen nachgebildet, die von einigen Programme zwingend benötigt werden. In jollix ist wine so eingerichtet, dass die Spiele und Programme, die in der jollix-wine-liste stehen, garantiert laufen. Eine deutschsprachige Seite über wine ist www.linux-wine.de, die Original Seiten sind auf www.winehq.com zu finden.

Was ist KDE?

KDE ist eine moderne grafische Arbeitsumgebung für UNIX Workstations. KDE strebt an, den Bedarf für eine einfach zu bedienende Arbeitsfläche für UNIX Workstations zu decken, ähnlich den Arbeitsumgebungen, die man unter MacOS oder Windows 9x/NT/2000/XP vorfindet. Alles Wissenswerte zu KDE gibt es auf www.kde.de nachzulesen.

Jollix Ausprobieren ohne Installation

Die LiveCD wird einfach ins CD-Rom oder DVD-Laufwerk des Rechners eingelegt. Im BIOS muss als erstes boot device "CDROM" o.ä. ausgewählt sein. Dann fährt jollix von der CD hoch und der Benutzer "user" wird automatisch angemeldet. Siehe auch die Readme.txt der aktuellen Version.

Installation

Noch kann jollix nicht installiert werden.
