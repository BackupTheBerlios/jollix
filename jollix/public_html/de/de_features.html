Jollix 0.1.1 Features:

  • basiert auf Gentoo Linux
  • zum Ausprobieren: keinerlei Installation auf Festplatte notwendig
  • für Athlon-CPUs optimiert (Pentium-Optimierung ist erstmal auf später verschoben, funktioniert aber auch mit Pentium II Rechnern aufwärts, i686)
  • unterstützt nVidia-basierte(volle 3D Beschleunigung) und alle anderen (nur Vesa!) Grafikkarten
  • Kernel: Gentoo 2.4.20-gaming-sources
  • KDE 3.1.1a
  • Winex-cvs 2.2.1
  • Spiel Deine Half-Life-, Public-Enemy- oder Counter-Strike Festplatteninstallation!
  • Multimedia Anwendungen (xmms, mplayer, xine u.a.)

Known Bugs / Bekannte Fehler in 0.1.1

  • Deutsches Tastaturlayout: ü,ö,ä und @-Zeichen vermurkst. (@ liegt auf ü)
  • Bei Problemen mit der Maus o.ä. im BIOS versuchsweise "Plug'n'Play OS" auf "NO" stellen

Geplante Änderungen und Features für die nächste Version:

  • ISDN-Unterstützung
  • Fritz!Card DSL
  • 3D Beschleunigung für ATI Grafikkarten
  • weitere Half-Life Mods
  • Quake3
  • Unreal Tournament 2003 (?)
  • fluxbox windowmanager (für schwache Rechner :-)
  • XQF, ein Game-Server Browser
  • aus Lizenzgründen wird WineX ersetzt durch das freie(re) Wine
  • Cheathing Death und HLSW Unterstützung
  • Teamspeak Client für Linux
  • Bugfixes + aktuelle Versionen der installierten Software

Was jolix irgendwann mal können soll:

  • auf Wunsch schnell und einfach installiert
  • beliebig erweiterbar, wie bie Gentoo (jollix _IST_ dann Gentoo ;-)

Was soll jollix können?

Im Forum könnt ihr Vorschläge machen, was jollix in zukünftigen Versionen alles mitbringen soll. Z.B. ein bestimmtes Programm, Unterstützung für ein bestimmtes Spiel, Half-Life Mod usw.
