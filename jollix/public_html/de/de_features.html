Jollix 0.3 Features:

  • basiert auf Gentoo Linux
  • LiveCD - Zum Ausprobieren ist keine Installation auf der Festplatte notwendig
  • Optimiert für i686 Architektur
  • Kudzu Hardware Erkennung
  • Linux Kernel: gentoo-dev-sources 2.6.5-r1
  • XOrg-X11 6.7.0
  • KDE 3.2.2
  • Viele Multimedia Anwendungen (audacity, xmms, xine, kino, kdenlive, K3B, u.a.)
  • WLAN und Bluetooth Support
  • Paketliste (in neuem Fenster)
