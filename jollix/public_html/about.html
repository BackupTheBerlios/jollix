Was ist jollix?

Jollix ist eine einfach zu installierende und komplett freie Linux Distribution für die x86 Architektur, die auf Gentoo Linux basiert. Mit jollix soll haupstächlich dem Gamer und Multimedia-Enthusiast der vollständige Umstieg von MS-Windows auf Linux erleichtert werden. Zum Ausprobieren kann man jollix - ganz wie Knoppix - direkt von einer CD booten, ohne die Festplatte mit bereits bestehenden Partitionen anzurühren. Dazu lädt man sich das ISO Image einfach herunter, brennt es auf eine CD und bootet damit den PC neu. Soll das bestehende System ersetzt werden, kann dieselbe CD zum Installieren eines vollwertigen und hochoptimierten Linux Systems verwendet werden.

Die Installation geschieht fast vollautomatisch, da sie bestimmte Hardwarekomponenten (Athlon CPU, nVidia oder ATi Grafiksystem, die gängigsten Soundkarten usw.) vorraussetzt. Für diese Umgebung ist jollix dann aber optimal konfiguriert und quasi "out-of-the-box" benutzbar!

Neben der Desktop-Umgebung KDE samt einiger nützlicher Programme (Multimedia-Anwendungen, CD-Brennprogramm usw.) ist auch wine enthalten. Wine ist ein Programmpaket, mit dem auch MS-Windows Programme unter Linux laufen. Dadurch sind insbesondere auch Spiele wie Counter-Strike unter Linux benutzbar, für die es (noch) keine Linux-Version gibt.

Sollte der Funktionsumfang irdendwann nicht mehr ausreichend sein, kann das System einfach erweitert werden. D.h. zusaetzliche Programmpakete können - ganz wie bei Gentoo Linux - installiert und auch wieder deinstalliert werden. Dafür wird das von den gentoo-Machern entwickelte Portage System gleich mit installiert. Jollix vereint so alle Vorteile der High-End-Geek-Linux Distribution Gentoo mit den Bedürfnissen und Ansprüchen eines MS-Windows Nutzers, den bisher nur das Zocken seines Lieblings-Shooters vom Umstieg auf Linux abgehalten haben.
