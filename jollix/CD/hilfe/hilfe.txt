0. Was ist jollix 0.2?
Jollix ist eine komplett freie Linux Distribution für die x86 Architektur,
die auf Gentoo Linux basiert. Mit jollix soll haupstächlich dem Gamer und
Multimedia-Enthusiast der vollständige Umstieg von MS-Windows auf Linux
erleichtert werden. Die Version 0.2 ist die zweite release von jollix.
Die wichtigsten Erweiterungen gegenüber der Version 0.1 ist die Unterstützung
von ATI Radeon Graphikkarten, ISDN und DSL Karte von AVM.


1. Netzwerk einrichten
Über "jollix einrichten" das Icon  "Netzwerk einrichten" anklicken, um das lokale
LAN-Netzwerk einzurichten. "eth0" bezeichnet die erste Netzwerkkarte im Rechner,
eth1 die zweite usw. Befindet sich nur eine Netzwerkkarte im Rechner, kann
die Vorgabe "eth0" übernommen werden.


2. ADSL einrichten
Die ADSL Verbindung wird eingerichtet über "ADSL einrichten".
Es öffnet sich ein Fenster, in dem das root-Passwort abgefragt wird.
Danach wird man durch die ADSL Einrichtung geführt.

Einige Anmerkungen:
a) T-Online DSL Kunden sollten den Benutzernamen wie folgt angeben:
1. Anschlusskennung
2. Zugehöriger T-Online Nummer
3. Suffix #0001
4. @t-online.de
z.B. 111111111111222222222222#0001@t-online.de
b) Befinden sich im Rechner mehr als eine Netzwerkkarte, gilt das gleiche
wie unter 2. "Netzwerk einrichten". Die erste Karte im Rechner ist "eth0",
die zweite "eth1" usw.

Sobald die Einrichtung abgeschlossen ist, kannst mit der Desktop-Verknüpfung
"ADSL starten" die Verbindung aufgebaut werden.


3. Fritz!CardDSL einrichten
Zur Einrichtung einer Fritz!CardDSL gibt es ein extra Icon unter 
"Jollix einrichten". Nach Klick auf "Fritz!CardDSL einrichten" gibt man hier
den vom Provider zugewiesenen Benutzernamen sowie das dazugehörige Passwort an.
Danach muss noch mindestens eine IP-Adresse vom Domain Name Service angegeben 
werden. Mehrere IP-Adressen werden durch Leerzeichen voneinander getrennt.
Standardmäßig ist 192.168.0.1 eingerichtet. Wenn z.B. ein eigener DNS-Service 
im LAN betrieben wird kann die Adresse des eigenen DNS-Servers hier gegebenenfalls 
angepasst werden.


4. ISDN einrichten
Es müssen die gleichen Angaben gemacht werden wie unter Punkt 3.
"Fritz!CardDSL einrichten".


5. Counterstrike einrichten
Über den Desktop-Button "Counterstrike einrichten" eine Desktop-Verknüpfung
mit der Counterstrike Installation auf der Festplatte einrichten.
Im erscheinenden Dialog den Installationsordner von Counterstrike angeben,
in dem sich hl.exe bzw. cstrike.exe befindet. Die Festplatten werden in
dem Dialog so oder ähnlich erscheinen: Partition1, Partition2 usw.
(Alle Partitionen sind schreibgeschützt und können von jollix nicht verändert
werden!)

6. Mausrad aktivieren
Es kann vorkommen, dass das Mausrad bei einigen Mäusen nicht auf Anhieb
funktioniert. Damit möglichst viele Mäuse, insbesondere auch Logitech-Mäuse,
funktionieren wurde jollix so vorkonfiguriert, das es unter Umständen notwendig
ist, die Mausrad-Unterstützung nachträglich zu aktivieren. 
Unter "Jollix einrichten" klickt man dazu das Icon "Mausrad (de)aktivieren" an.
Nachdem man seine Wahl getroffen hat muss der X-Server neu gestartet werden.
Das geht am einfachsten, indem man im laufenden KDE die Tastenkombination
STRG+ALT+BACKSPACE drückt.
ACHTUNG! Bei aaktiviertm Mausrad kann es zu Fehlfunktionen bei einigen
Logitech-Mäusen kommen!


4. Passwörter
Damit es fürs erste einfacher ist, sind die Passwörter für die beiden
eingerichteten Benutzer vorgegeben. Der Benutzer "root" ist vergleichbar mit dem
"Administrator" unter Windows, daher sollte man nur dann "root" benutzen, wenn es
von jollix ausdrücklich verlangt wird. Der Benutzer "user" ist der eingerichtete
Standardbenutzer, mit dem man spielen und surfen sollte. Vor der Einrichtung eines
Internetzuganges unter jollix (Netzwerk oder ADSL), sollten die beiden Passwörter
unbedingt geändert werden! Nach einem Neustart der jollix LiveCD sind die beiden
Passwörter allerdings wieder zurückgesetzt, da sie nicht dauerhaft auf der CD gespeichert
werden können.
Das root-Passwort und das user-Passwort ändert man am einfachsten, indem man
in "jollix einrichten" die "Benutzerverwaltung" aufruft und dort in der Liste den Benutzer
"user" bzw. "root" per Doppelklick auswählt.

Das root-Passwort lautet: jollix

Das user-Passwort lautet: user
